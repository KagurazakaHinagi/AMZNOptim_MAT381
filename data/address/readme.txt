See https://git-lfs.github.com/ for more information.

Data Source: https://openaddresses.io/

Data Explanation:
The data contains some valid addresses in the King County of Washington state.
The data is in geojson format, see https://geojson.org/ for more information.

When using this data, filter for zipcode 980xx, 981xx, and 982xx to get the addresses only in the Greater Seattle Area.
